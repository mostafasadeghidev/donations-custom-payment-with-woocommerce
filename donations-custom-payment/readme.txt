== Description ==

**Donations & Custom Payment** is a comprehensive WooCommerce donation and custom payment plugin that creates flexible payment forms with custom amounts. Perfect for charities, non-profits, crowdfunding campaigns, and businesses that want to offer pay-what-you-want options.

**Key Features:**
* **WooCommerce Donation System**: Seamless integration with WooCommerce for secure payments
* **Custom Payment Forms**: Create flexible payment forms with custom amount inputs
* **Pay What You Want**: Allow customers to choose their own payment amount
* **Flexible Payment Options**: Set minimum and maximum amounts with preset buttons
* **Donation Management**: Track and manage all donations through WooCommerce
* **Multi-language Support**: Fully translatable with Persian and English included
* **Color Customization**: Complete control over colors with alpha transparency
* **Admin Reports**: Track orders and sales with filtering options
* **Mobile Responsive**: Works perfectly on all devices

= Usage =

Simply use the shortcode `[custom_payment_form]` anywhere you want to display the payment form.

= Languages =

* Persian (Farsi) - fa_IR
* English - en_US

= Requirements =

* WordPress 5.0 or higher
* WooCommerce plugin
* PHP 7.4 or higher

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/donations-custom-payment` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Use the Settings->Custom Payment Form screen to configure the plugin
5. Add the shortcode `[custom_payment_form]` to any page or post

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, this plugin requires WooCommerce to be installed and activated for the checkout functionality.

= Can I customize the colors? =

Yes! The plugin includes advanced color customization with transparency support for all elements.

= Is the plugin translatable? =

Yes, the plugin is fully translation-ready and includes Persian and English translations out of the box.

= Can I track orders? =

Yes, the plugin includes a comprehensive reports section showing all orders with filtering options.

= Does it work on mobile devices? =

Yes, the plugin is fully responsive and works perfectly on all devices.

= Is the plugin compatible with WooCommerce HPOS? =

Yes, the plugin is fully compatible with WooCommerce High-Performance Order Storage (HPOS) feature.
